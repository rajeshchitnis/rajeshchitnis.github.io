List of FPT+Approximation results

Please send any comments/corrections/submissions to fptapprox@gmail.com



  • Strongly Connected Steiner Subgraph: Given a digraph \(G=(V,E)\) and a set of terminals \(T=\{t_1, t_2, \ldots, t_k\}\) is there a set \(H\subseteq V\) of size \(\leq p\) such that \(G[H]\) has an \(t_i \leadsto t_j\) path for each \(1\leq i\neq j\leq k\)
    Algorithm: 2-approximation in \(2^{k}\cdot n^{O(1)}\) time [Reference]
    Lower Bound: No \((2-\epsilon)\)-approximation in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH [Reference]

  • Directed Odd Cycle Transversal: Given a digraph \(G=(V,E)\) is there a set \(H\subseteq V\) of size \(\leq k\) such that \(G[V\setminus H]\) has no (directed) odd cycles
    Algorithm: 2-approximation in \(2^{O(k^2)}\cdot n^{O(1)}\) time [Reference]
    Lower Bound: There is a constant \(\epsilon>0\) such that there is no \((1+\epsilon)\)-approximation in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Parameterized Inapproximability Hypothesis (PIH) and \(FPT\neq W[1]\) [Reference]

  • Directed Steiner Network: Given a digraph \(G=(V,E)\) and a set of terminal pairs \(T=\{(s_i, t_i)\ :\ i\in [k]\}\) is there a set \(H\subseteq V\) of size \(\leq p\) such that \(G[H]\) has an \(s_i \leadsto t_i\) path for each \(i\in [k]\)
    Algorithm: \(k^{1/2+\epsilon}\)-approximation in polynomial time for every \(\epsilon>0\) time [Reference 1] [Reference 2]
    Lower Bound: No \((k^{1/4-o(1)})\)-approximation in \(f(k)\cdot n^{O(1)}\) time (for any function \(f\)) under Gap-ETH [Reference]
